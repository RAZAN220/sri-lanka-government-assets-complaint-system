# Sri Lanka Government Assets Complaint Management System

A modern web-based **Government Assets Complaint Management System** developed to help citizens report issues related to public infrastructure across **Sri Lanka's 9 Provinces and 25 Districts**. The system enables government authorities to efficiently manage, monitor, and resolve complaints through a role-based platform.

---

## 📖 Project Overview

The **Sri Lanka Government Assets Complaint Management System** is designed to improve communication between citizens and government authorities by providing a centralized platform for reporting and tracking complaints related to public assets.

Citizens can submit complaints with photos and location details, while district and provincial officers manage and resolve issues through dedicated dashboards. Super administrators oversee the entire system with reports and analytics.

---

## 🎯 Objectives

- Digitize the public complaint management process.
- Improve response time for government authorities.
- Provide transparency through complaint tracking.
- Manage complaints across all provinces and districts.
- Generate reports and analytics for better decision-making.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/RAZAN220/sri-lanka-government-assets-complaint-system.git
```

2. Move the project to your XAMPP `htdocs` folder.

3. Import the SQL database into phpMyAdmin.

4. Update database credentials in:

```
config/database.php
```

5. Start Apache and MySQL.

6. Open your browser and visit:

```
http://localhost/sri-lanka-government-assets-complaint-system/
```
